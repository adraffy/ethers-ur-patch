# ethers-ur-patch

Monkeypatch for [ENSIP-19](https://docs.ens.domains/ensip/19) and [ENSIP-23](https://docs.ens.domains/ensip/23) support in [ethers.js](https://github.com/ethers-io/ethers.js/) v5 and v6.

#### Roadmap 

* Downgrade versions to lowest supported
* Explore normalization options for v5
* Build 2 libraries
* Add tests for checking patched return types
* Add tests for failures
